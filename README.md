## Selected Projects

### Attijariwafa Bank Data Pipeline & Analytics
Built a comprehensive data pipeline (NiFi → HDFS → Spark → PostgreSQL → Grafana) to centralize sales data, implement Random Forest models, detect anomalies, and generate interactive dashboards for marketing insights.  
[View Repository](https://github.com/mreckah/EcomPipeline)

### Amazon Delivery Data Pipeline
Developed a distributed data pipeline with 3 HDFS DataNodes to process raw delivery data through Bronze, Silver, and Gold layers. Implemented an AI model to predict delivery times and analyze agent performance and traffic patterns. Results stored in PostgreSQL with Grafana dashboards for KPI monitoring.  
[View Repository](https://github.com/mreckah/DeliveryPipeline)

### Remedy - AI Healthcare Assistant
Intelligent healthcare application providing personalized skin health analysis through natural language conversation. Built with custom ResNet50 computer vision model, Llama-3.3-70b language model, and Whisper AI for voice interaction. Features automated professional PDF medical report generation and ChromaDB for medical knowledge retrieval.  
[View Repository](https://github.com/mreckah/medical-assistant)

### Network Anomaly Detection System (ES Atlas Cloud Services)
Machine learning system detecting network anomalies using Isolation Forest and K-Means algorithms, classifying DDoS attacks, brute force attempts, and port scanning activities.

### Souk Platform (3rd Place, Hackathon)
Inclusive e-commerce platform integrating AI chatbot functionality and accessibility-focused navigation design.
